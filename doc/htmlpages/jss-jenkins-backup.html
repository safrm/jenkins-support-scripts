Name

jss-jenkins-backup — Basic and simple jenkins backup

Synopsis

jss-jenkins-backup

DESCRIPTION

Basic and simple jenkins backup. Using specific fileter BACKUP_FILTER (set from script) creates /opt/jenkins-backup/manual-jss/ <timestamp>.tar.bz2 backup.
